# Snake Water Gun Game

A command-line implementation of the Snake Water Gun game, similar to Rock Paper Scissors.

## Rules

- Snake drinks water (Snake wins against Water)
- Water rusts gun (Water wins against Gun)
- Gun shoots snake (Gun wins against Snake)

## How to Run

1. Make sure you have Node.js installed.
2. Install dependencies: `npm install`
3. Run the game: `node index.js`
4. Enter your choice: 'snake', 'water', or 'gun'
5. See the result against the bot's random choice.

## Technologies Used

- Node.js
- prompt-sync (for user input)
